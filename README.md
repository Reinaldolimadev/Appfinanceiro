# 💰 Finance.AI - Educador Financeiro Inteligente

Aplicação desenvolvida durante o Bootcamp Santander 2026 da DIO.

O objetivo do projeto é auxiliar usuários no planejamento financeiro por meio de Inteligência Artificial, gerando diagnósticos personalizados, sugestões financeiras e permitindo conversar com um Educador Financeiro virtual.

---

🚀 Funcionalidades

📋 Simulação Financeira

- Cadastro de renda mensal
- Cadastro de gastos fixos
- Cadastro de dívidas
- Definição de objetivo financeiro
- Valor da meta
- Prazo para alcançar a meta

---

🤖 Diagnóstico com Inteligência Artificial

Após preencher a simulação, a aplicação utiliza IA para gerar:

- Análise de viabilidade da meta
- Diagnóstico financeiro
- Sugestões de economia
- Sugestões de renda extra
- Sugestões de investimentos
- Mensagem motivacional personalizada

---

💬 Chat com IA

O usuário pode conversar com o Educador Financeiro sobre sua própria simulação.

Funcionalidades:

- Perguntas ilimitadas
- Respostas contextualizadas
- Histórico da conversa
- Persistência das conversas no LocalStorage
- Scroll automático
- Loading durante respostas
- Tratamento de erros

---

📚 Histórico de Simulações

- Visualizar todas as simulações
- Ver detalhes de uma simulação
- Reutilizar insights já gerados
- Excluir simulações
